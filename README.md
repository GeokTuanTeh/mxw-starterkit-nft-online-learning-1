# Installation



Steps:
===========
npm init

npm install mxw-sdk-js --save

prepare index.ts

prepare tsconfig.json



Documentation:
==================
1. Run `npm install`
2. create 2 empty folders under folder docs:
   _static
   _templates
3. Run `make build`
4. Run `docker run --rm -p 8080:80 mxw-sdk-js-doc` to start service
5. Open URL http://localhost:8080/ to view doc
